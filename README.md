# 🎮 GamerZone Store - Landing Page Responsive

¡Bienvenido al repositorio de **GamerZone Store**! Este proyecto consiste en una Landing Page moderna, intuitiva y completamente adaptativa (responsive), diseñada para centralizar la información de una tienda especializada en productos y accesorios para videojuegos de alta calidad.

## 📝 Descripción del Proyecto

Actualmente, los jugadores y creadores de contenido sufren al tener que buscar en múltiples tiendas o redes sociales para comparar precios, revisar especificaciones técnicas o cazar ofertas. **GamerZone Store** resuelve este problema centralizando todo en una sola plataforma web optimizada para dispositivos móviles.

La página web está diseñada bajo un concepto visual que transmite **tecnología, innovación y alto rendimiento**, permitiendo a los usuarios explorar el catálogo de productos, conocer las marcas aliadas y realizar consultas directas a través de un formulario dinámico.

---

## 👥 Segmentos de Usuario Atendidos

La plataforma está optimizada para cubrir las necesidades específicas de cuatro perfiles clave:

*   **🕹️ Gamers Casuales:** Usuarios que buscan periféricos estándar y accesorios accesibles para mejorar sus sesiones de juego del día a día.
*   **🔥 Gamers Competitivos y Streamers:** Entusiastas del hardware que demandan periféricos de alto rendimiento (teclados mecánicos, ratones con sensores de alta precisión, monitores de alta tasa de refresco y sillas ergonómicas).
*   **🎥 Creadores de Contenido:** Profesionales que requieren dispositivos de captura de video, micrófonos de estudio y accesorios especializados para transmisiones en vivo.
*   **🏢 Empresas e Instituciones Educativas:** Clientes corporativos que buscan equipar salas gamer, laboratorios de cómputo o espacios de entretenimiento.

---

## 🛠️ Tecnologías Utilizadas

Para el desarrollo de esta Landing Page se optó por una arquitectura ligera y nativa, priorizando la velocidad de carga y la compatibilidad móvil:

*   **HTML5:** Estructura semántica del sitio web para garantizar una excelente accesibilidad y bases sólidas de SEO.
*   **CSS3:** Estilos personalizados utilizando técnicas modernas de diseño adaptativo (como Flexbox y CSS Grid) con una estética futurista y "gamer" (tonos oscuros con acentos de luces neón).
*   **JavaScript (ES6+):** Programación de interactividad en el sitio, animaciones ligeras y la **lógica de validación en tiempo real para el formulario de contacto** (incluyendo una robusta verificación de seguridad en el campo de contraseña).

---

## 🔒 Características Destacadas: Validación de Contraseña

El formulario de contacto implementa un script de JavaScript dedicado a validar que las contraseñas ingresadas por los usuarios cumplan con los estándares de seguridad modernos antes de permitir el envío del formulario. El script verifica activamente:
1. Una longitud mínima de caracteres.
2. La inclusión de números y letras.
3. El uso de caracteres especiales para mitigar riesgos de seguridad.
